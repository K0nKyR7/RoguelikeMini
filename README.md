## ✨ Особенности

- **🎲 Полная процедурная генерация:** Каждое прохождение уникально. Уровни создаются с использованием алгоритмов BSP-деревьев и клеточных автоматов.
- **👤 Глубокая кастомизация персонажа:** 4 класса (Рыцарь, Вор, Колдун, Следопыт) с уникальными стартовыми навыками и древом развития.
- **⚔️ Тактические пошаговые бои:** Учитывайте поле зрения, укрытия, ловушки и особенности поведения врагов.
- **🛡️ Система экипировки:** Слоты для оружия, брони, колец и артефактов. Более 50 видов предметов с особыми свойствами.
- **🏙️ Мирные локации:** Город **Крестгард** с таверной, рынком, кузницей и храмом для отдыха и подготовки между вылазками.
- **👹 Разнообразный бестиарий:** Гоблины, гигантские пауки, скелеты и уникальные боссы с особыми фазами сражения.
- **🐍 Чистый Python:** Минимум внешних зависимостей. Игра использует только библиотеку `tcod` для отрисовки псевдографики в терминале.

## 📜 Сюжет

Граница мира истончилась, и древнее королевство магов восстало из-под земли, превратившись в **Бездну Тысячи Склепов**. Из глубин лезут чудовища, а в воздухе витает запах древнего проклятия.

Вы — искатель приключений, прибывший в форпост **Крестгард**. Ваша цель сокрыта в самом сердце подземелья. Это может быть:
- **Корона Отверженных** — последняя надежда на исцеление смертельной болезни.
- **Кристалл Памяти** — ключ к вашему забытому королевскому прошлому.
- **Золото и Слава** — ведь чем глубже вы копаете, тем ценнее добыча.

Спуститесь на **50 этажей**, одолейте **5 древних боссов** и измените судьбу мира — или умрите в темноте, став очередным скелетом в бесконечных коридорах.

## 🚀 Быстрый старт

### Требования
- **Python 3.8** или новее
- **pip** (менеджер пакетов Python)
- **Git** (для клонирования репозитория)

### Запуск игры (релизная версия)

1. **Клонируйте репозиторий:**
   ```bash
   git clone https://github.com/yourusername/python-rl-roguelike.git
   cd python-rl-roguelike
Установите зависимости:

bash
pip install -r requirements.txt
Основная зависимость: tcod>=16.0.0

Запустите игру:

bash
python main.py
🎮 Управление
Управление осуществляется с клавиатуры. Поддерживаются два режима: Numpad и Vim Keys (HJKLYUBN).

Действие	Клавиша (Numpad)	Клавиша (Vim)	Описание
Движение	1, 2, 3, 4, 6, 7, 8, 9	B J N H L Y K U	Перемещение на 1 клетку
Ожидание	5 или .	.	Пропустить ход
Подобрать предмет	g или ,	g	Взять предмет с пола
Инвентарь	i	i	Открыть рюкзак
Экипировка	e	e	Меню смены снаряжения
Спуск по лестнице	>	>	Переход на следующий этаж
Осмотреться	x	x	Режим осмотра (look mode)
Меню персонажа	c	c	Лист характеристик
Выход из игры	Esc	Esc	Сохранить и выйти
Горячие клавиши способностей:

F1 - Использовать классовое умение.

z - Использовать свиток/зелье (вызов меню).

⚙️ Игровые механики
Создание персонажа
При старте новой игры вам будет предложено выбрать класс и предысторию. Это определит ваши начальные характеристики (Сила, Ловкость, Интеллект, Телосложение) и уникальную способность.

Класс	Способность	Стиль игры
🛡️ Рыцарь	Блок: Снижает входящий урон на 50% на 1 ход.	Танкование, выживание в гуще врагов.
🗡️ Вор	Удар в спину: Критический урон x3 по невидящей цели.	Скрытность, маневрирование, высокий DPS.
🔥 Колдун	Огненный шар: Урон по площади (AoE).	Контроль толпы, дистанционный магический урон.
🏹 Следопыт	Метка охотника: Следующая атака игнорирует броню цели.	Снайперская стрельба, уничтожение приоритетных целей.
Инвентарь и Экипировка
Вместимость: 26 слотов (a-z). Предметы можно бросать d или использовать u.

Слоты снаряжения:

Main Hand (Оружие)

Off Hand (Щит или второе оружие)

Head (Шлем)

Chest (Нагрудник)

Legs (Поножи)

Ring 1 / Ring 2 (Кольца)

Город Крестгард
После каждой успешной вылазки (или побега) вы возвращаетесь в город, который служит хабом:

Таверна: Восстановление HP и MP, найм спутников.

Кузница: Ремонт сломанного оружия, улучшение брони до +3.

Рынок: Покупка расходников.

Храм: Снятие проклятий и идентификация магических вещей.

🛠️ Установка для разработки
Если вы хотите внести свой вклад в код или модифицировать игру:

Установите виртуальное окружение:

bash
python -m venv venv
source venv/bin/activate  # Linux/Mac
# или
venv\Scripts\activate  # Windows
Установите зависимости разработчика:

bash
pip install -r requirements-dev.txt
Сюда входят pytest для тестов и black для форматирования кода.

Запуск тестов:

bash
pytest tests/
🎯 Как играть (Советы новичкам)
Не спешите. Это пошаговая игра. Оценивайте обстановку каждый ход.

Используйте коридоры. Сражайтесь в дверных проемах, чтобы враги нападали на вас по одному.

Идентифицируйте зелья в безопасном месте. Выпитое у костра зелье яда не убьет вас мгновенно.

Слушайте шум. В интерфейсе отображаются звуки шагов врагов за пределами поля зрения.
